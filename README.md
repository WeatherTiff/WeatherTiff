# WeatherTiff - Professional Weather App

A modern, production-ready weather application with real-time data, hourly forecasts, 7-day predictions, and severe weather alerts.

## Features

✨ **Real-Time Weather Data** - Live current conditions with Open-Meteo API  
⏰ **Hourly Forecast** - 12-hour detailed hourly weather  
📅 **7-Day Forecast** - Extended forecast with high/low temperatures  
📍 **Multiple Locations** - Save and switch between unlimited locations  
⚠️ **Severe Weather Alerts** - Real-time storm and severe weather warnings  
🎨 **Modern UI** - Material Design 3 with smooth animations  
📱 **Fully Responsive** - Works perfectly on mobile, tablet, and desktop  
🌙 **Dark/Light Theme** - Adaptive theme toggle  

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy to Vercel (Recommended)

### Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Create a repository named `weathertiff`
3. Follow the instructions to push your code

### Step 2: Deploy on Vercel

1. Go to https://vercel.com
2. Click "New Project"
3. Import your GitHub repository
4. Click "Deploy"
5. Done! Your app is live 🎉

**Your app will be available at:** `https://weathertiff.vercel.app` (or a custom URL you choose)
